# 排序算法可视化

一个用 Python 实现的排序算法可视化工具，包含快速排序和冒泡排序。

## 功能特点

- 🚀 **快速排序** - 高效的分治算法
- 🫧 **冒泡排序** - 经典的交换排序算法
- 📊 **实时可视化** - 动态展示排序过程
- 🎨 **彩色显示** - 不同颜色标识比较、交换和已排序元素

## 安装依赖

```bash
pip install matplotlib
```

## 使用方法

```bash
python sorting_visualizer.py
```

## 算法说明

### 快速排序 (Quick Sort)
- **时间复杂度**: 平均 O(n log n)，最坏 O(n²)
- **空间复杂度**: O(log n)
- **特点**: 分治策略，效率高

### 冒泡排序 (Bubble Sort)
- **时间复杂度**: O(n²)
- **空间复杂度**: O(1)
- **特点**: 简单直观，适合小规模数据
